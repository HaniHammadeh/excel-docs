# Data Analysis with Excel

Excel provides powerful tools to analyze and visualize data. Here are some of the key features:

## Sorting Data

You can sort data by columns, either in ascending or descending order.

1. Select the column you want to sort.
2. Go to the `Data` tab.
3. Click `Sort Ascending` or `Sort Descending`.

## Filtering Data

Filtering allows you to display only the rows that meet certain criteria.

1. Select the data range.
2. Click on the `Data` tab.
3. Click `Filter` to add drop-down menus to each column header.
4. Use the drop-downs to filter your data.

## Pivot Tables

Pivot tables allow you to summarize large amounts of data quickly.

1. Select your data range.
2. Go to the `Insert` tab and click `Pivot Table`.
3. Choose where to place the pivot table and what data to include.
