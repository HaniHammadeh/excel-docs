# Excel Keyboard Shortcuts

These shortcuts will help you save time and be more efficient while using Excel:

## Basic Shortcuts

- `Ctrl + C` – Copy
- `Ctrl + V` – Paste
- `Ctrl + Z` – Undo
- `Ctrl + Y` – Redo

## Navigation Shortcuts

- `Ctrl + Arrow Keys` – Move to the edge of data
- `Ctrl + Home` – Move to the top-left of the worksheet
- `Ctrl + End` – Move to the bottom-right of the worksheet

## Formula Shortcuts

- `Alt + =` – Automatically sum selected cells
- `Ctrl + Shift + Enter` – Enter an array formula
- `F4` – Repeat the last action
